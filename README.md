# Sales Data Analysis & AI Insights Platform

A full-stack data analytics + machine learning project built using
Streamlit, Supabase, and ML models.

------------------------------------------------------------------------

## Features

-   Sales Dashboard
-   Payment Risk Prediction
-   Customer Segmentation
-   Revenue Forecasting
-   Investment Insights
-   Product Simulation

------------------------------------------------------------------------

## Tech Stack

-   Streamlit
-   Python
-   Supabase (SQL)
-   Scikit-learn
-   Pandas, NumPy

------------------------------------------------------------------------

## Local Installation

``` bash
git clone https://github.com/VedanshiBarot/sales_analytics_ml_dashboard.git
cd sales_analytics_ml_dashboard
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run "streamlit app/app.py"
```
