# 📊 Sales Data Analysis & AI Insights Platform

A full-stack data analytics + machine learning project built using
Streamlit, Supabase, and ML models.

------------------------------------------------------------------------

## 🚀 Features

-   📊 Sales Dashboard\
-   💳 Payment Risk Prediction\
-   👥 Customer Segmentation\
-   📉 Revenue Forecasting\
-   💼 Investment Insights\
-   🧪 Product Simulation

------------------------------------------------------------------------

## 🧠 Tech Stack

-   Streamlit\
-   Python\
-   Supabase (PostgreSQL)\
-   Scikit-learn\
-   Pandas, NumPy

------------------------------------------------------------------------

## ⚙️ Local Installation

``` bash
git clone https://github.com/your-username/sales-analysis-project.git
cd sales-analysis-project
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run "streamlit app/Home.py"
```

------------------------------------------------------------------------

## 🌐 Deployment (Streamlit Cloud)

1.  Push code to GitHub\
